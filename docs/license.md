# License

This page describes the intended reuse terms for Missionary Readiness Institute public website content. It is a starter policy and should be finalized before public launch.

## Recommended Default

Recommended content license: `Creative Commons Attribution-NonCommercial 4.0 International (CC BY-NC 4.0)`

This default would allow churches, mentors, and training partners to share and adapt public training content for noncommercial ministry use, provided they give appropriate credit and do not imply endorsement.

## Suggested Attribution

When reusing material, attribute it as:

> Adapted from Missionary Readiness Institute, `https://missionary-readiness-institute.github.io/readiness-platform/`

## Not Covered By This License

The public content license should not automatically apply to:

- Logos, trademarks, branding, or organization names.
- Private training materials not published on this site.
- Third-party content quoted or linked from the site.
- Contributor content governed by a separate agreement.

## Code And Site Configuration

Repository code, MkDocs configuration, and automation may use a different software license if the organization chooses one. Add that license here or in a root-level `LICENSE` file when finalized.

!!! warning "License decision needed"
    Confirm the final content and code licenses with the responsible organization before treating this page as authoritative.
