# Course Catalog

The course catalog introduces foundational readiness tracks for prospective missionaries and the people preparing them. Each course page is currently a starter outline that can be expanded into lessons, discussions, readings, reflection prompts, and assessments.

## Foundational Tracks

| Course | Purpose |
| --- | --- |
| [Spiritual Formation](courses/spiritual-formation.md) | Prepare learners for worshipful, resilient, and obedient service. |
| [Cross-Cultural Service](courses/cross-cultural-service.md) | Build humility, listening skills, and cultural learning habits. |
| [Missionary Care](courses/missionary-care.md) | Help learners plan for relational health, support, and sustainability. |
| [Language Learning](courses/language-learning.md) | Introduce realistic expectations and habits for language acquisition. |
| [Field Readiness](courses/field-readiness.md) | Prepare for practical field entry, logistics, communication, and risk awareness. |

## Suggested Use

Churches and mentors can use these tracks as conversation guides before formal deployment or field-specific onboarding. Learners can also work through the catalog privately, but the material is strongest when paired with trusted local counsel.

!!! caution "Not a certification"
    Completion of these pages does not certify a learner for missionary service. Sending decisions should remain with the learner's church, ministry organization, and appropriate leaders.
