# Language Learning

## Purpose

Introduce realistic language-learning expectations and practical habits for learners preparing to communicate in a new context.

## Learning Objectives

By the end of this course, learners should be able to:

- Set realistic expectations for language acquisition.
- Describe daily habits that support progress.
- Practice humility when communication is slow or uncomfortable.
- Plan for language learning as ministry preparation, not a side task.

## Starter Modules

1. Why language learning matters.
2. Goals, stages, and realistic timelines.
3. Listening-first practice.
4. Local helpers, tutors, and conversation partners.
5. Perseverance through embarrassment and fatigue.

## Reflection Prompt

How might language learning reshape your posture toward the people you hope to serve?
