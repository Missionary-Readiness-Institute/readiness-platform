# The E-Scale Ministry Assessment

## Purpose
A worksheet to help independent workers define the cultural distance of their intended host community and assess preparation gaps.

## How to use this assessment

1. Place your calling on the E-Scale from E-0 to E-3.
2. Describe the cultural distance and likely challenges.
3. Identify preparation gaps in language, worldview, and support.

## Assessment template

| E-Scale level | Description | Your context | Preparation needed |
| --- | --- | --- | --- |
| E-0 | Nominal Christians in own culture | | |
| E-1 | Unchurched people in same culture | | |
| E-2 | Similar but different culture | | |
| E-3 | Vastly different culture | | |

## Notes

- The higher the E-Scale, the greater the need for culture learning and language acquisition.
- Use this assessment with a mentor or sending team for feedback.
