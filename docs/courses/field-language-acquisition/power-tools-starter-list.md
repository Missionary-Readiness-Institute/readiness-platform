# The "Power Tools" Starter List

## Purpose

A quick worksheet for learners to translate 20 essential phrases with a Language Helper.

## Instructions

Translate each phrase into the target language, practice it aloud, and use it in the next helper session.

## Power tool phrases

1. Please repeat.
2. Say it slower.
3. Correct my mistake.
4. Speak only in the target language.
5. What does this mean?
6. How do you say that?
7. Can I try it again?
8. Listen to me.
9. I do not understand.
10. I understand a little.
11. Please write it down.
12. Show me again.
13. Is that right?
14. Please ask me.
15. Help me say this.
16. Say it with me.
17. One more time.
18. What did you say?
19. Please wait.
20. Thank you.

## Notes

- Use the list to keep sessions active and learner-centered.
- Add additional phrases that are important for your context.
- Review the list before every language practice meeting.
