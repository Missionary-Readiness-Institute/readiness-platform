# Local Proverb & Idiom Tracker

## Purpose
Document local proverbs and idioms to uncover cultural logic, values, and windows for Gospel conversation.

## How to use this tracker

1. Record the exact proverb or idiom.
2. Ask a local source for its literal meaning and common usage.
3. Describe the values and consequences it reflects.
