# Spiritual Players Profile

## Purpose
Capture and map local spiritual authorities, traditional healers, and faith influencers so workers can understand who shapes community belief.

## How to use this profile

1. Identify one spiritual player in the local community.
2. Record their role, influence, and claims.
3. Note how their authority interacts with local priorities.
