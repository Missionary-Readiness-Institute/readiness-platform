# The 7 S's Member Care Audit

## Purpose

A diagnostic tool for ministries to evaluate their support systems across the seven member care domains.

## Instructions

Rate each item from 1 (weak) to 5 (strong). Note gaps and assign follow-up actions.

## Audit table

| S | Description | Current rating (1-5) | Strengths | Gaps | Next action |
| --- | --- | --- | --- | --- | --- |
| Savior care | Discipleship, Spiritual Readiness and Resilience, and mission clarity | | | | |
| Self-care | Personal health, rest, and boundaries | | | | |
| Staff care | Peer support, mutual encouragement, team rhythms | | | | |
| Shepherd care | Pastoral oversight, counseling, mentoring | | | | |
| Structure care | Policies, logistics, safety systems | | | | |
| Sender/Supporter care | Sending church, family, and supporter relationships | | | | |
| Specialist care | Access to clinical, medical, or trauma professionals | | | | |

## Notes

- Use this audit annually and after any major incident.
- Include field workers, sending leaders, and care specialists in the review.
- Treat low ratings as opportunities for system-building rather than blame.
