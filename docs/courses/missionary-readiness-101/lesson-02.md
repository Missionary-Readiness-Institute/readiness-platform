# Lesson 02: Readiness Areas & Risk Awareness

## Key Idea

Readiness is a whole-task habit. Spiritual, relational, cultural, safety, and practical preparation each reduce different kinds of risk.

## Lesson Goal

Map the main readiness areas and recognize the risks when one area is ignored.

## What This Lesson Teaches

- Readiness is a whole-task habit. Spiritual, relational, cultural, safety, and practical preparation each reduce different kinds of risk.
- How the idea connects to the course output.
- A practical action you can complete immediately.

## Practical Action

Write one sentence for each readiness area and note one immediate question you need to answer in each area.
