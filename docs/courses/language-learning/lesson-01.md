# Lesson 01: Language Learning As Love

Language learning is more than a practical skill. It is a way to honor people, listen carefully, and enter community with humility.

## Key Idea

Learning a language is a long act of respect. It helps workers serve as guests and learners rather than as distant outsiders.

## Core Practices

### Set Realistic Expectations

Language learning takes time. Learners should expect embarrassment, fatigue, repeated mistakes, and slow progress.

### Listen First

Listening builds pronunciation, rhythm, vocabulary, and cultural understanding. Early listening is not wasted time.

### Use Local Helpers

Tutors, neighbors, church members, and conversation partners can help learners practice in real settings.

### Practice Daily

Small daily habits usually help more than occasional intense effort.

## Reflection

1. What makes language learning intimidating?
2. Who could help you practice?
3. What daily rhythm is realistic this month?

## Lesson Summary

Language learning is part of faithful presence. It helps workers listen, respect, and serve more personally.
