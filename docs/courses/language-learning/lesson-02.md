# Lesson 02: Listening-First Habits

## Key Idea

Listening builds understanding before speaking. It helps learners notice patterns, sounds, and rhythm in a new language.

## Lesson Goal

Establish a listening-first approach to language learning.

## What This Lesson Teaches

- Listening builds understanding before speaking. It helps learners notice patterns, sounds, and rhythm in a new language.
- How the idea connects to the course output.
- A practical action you can complete immediately.

## Practical Action

Choose one listening habit and practice it for one week, then note what you learned.
