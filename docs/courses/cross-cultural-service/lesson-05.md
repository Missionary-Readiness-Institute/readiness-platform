# Lesson 05: Reproducible Group Habits

## Key Idea

A movement only grows when practices are simple enough for others to reproduce without the original trainer.

## Lesson Goal

Build habits that others can repeat quickly and faithfully.

## What This Lesson Teaches

- A movement only grows when practices are simple enough for others to reproduce without the original trainer.
