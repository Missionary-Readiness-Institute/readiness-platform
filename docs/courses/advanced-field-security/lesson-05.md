## Creative-Access Digital Security Checklist

::: admonition warning
- [ ] **Pseudonyms:** Use pen names on all public communication and newsletters. NEVER associate a pseudonym with a real photo of the worker or local contacts.
- [ ] **Hardware:** Use stripped-down travel laptops with encrypted hard drives when crossing high-risk borders. Do not carry sensitive local contact lists.
- [ ] **Networks:** Avoid public WiFi. Use reputable VPN services where legal and normal.
- [ ] **Email:** Always use BCC for group emails. Avoid all incriminating and unwise terminology (religious jargon that triggers surveillance).
- [ ] **Photos:** Remove metadata (Exif data) from digital photos before uploading them to prevent geographic tracking.
:::
