# DMLC 90-Day Action Plan

## Purpose
A template for regional teams to set SMART goals, practice regularly, and hold each other accountable in a Disciple-Making Learning Community.

## How to use this template

1. Define a 90-day learning objective.
2. Set practice cycles and weekly review meetings.
3. Track progress with measurable activity and results.

## Weekly rhythm

- Week 1: Practice storying and report results
- Week 2: Review field experiments and adjust
- Week 3: Share lessons in the DMLC
- Week 4: Set next-cycle goals

## Notes

- Keep the ratio 3 parts practice to 1 part theory.
- Focus on field experiments, not ideal plans.
- Use results to guide the next 90-day cycle.
